售后流程说明
标题：售后支持流程

联系客服
通过电话、电子邮件或在线聊天联系我们。
提交请求
提供订单号和遇到的问题描述。
问题评估
我们的团队将在24小时内回复并提供解决方案。
解决方案实施
根据具体情况，可能涉及产品更换、维修或退款。
常见问题 FAQ
标题：常见问题解答

问：我收到的产品有损坏怎么办？
答：请立即拍照并通过我们的客户服务渠道联系我们，我们将尽快为您提供解决方案。
问：我可以取消我的订单吗？
答：在产品发货前，您可以随时取消订单。请联系客户服务以确认状态。
退货政策
标题：退货政策

所有退货必须在购买后30天内申请。
商品必须保持原始状态且未使用。
特定商品（如定制品）不接受退货，请参阅产品详情页面了解更多信息。
服务条款
标题：服务条款

适用范围
这些条款适用于所有通过我们平台进行的交易。
用户责任
用户需对其账户下的所有活动负责。
隐私保护
我们承诺保护用户的个人信息，并仅在获得许可的情况下使用。
示例工单流程
标题：工单处理流程

创建工单
登录您的账户，选择“帮助中心”，然后点击“新建工单”。
填写详细信息
包括问题描述、订单编号等相关信息。
等待响应
我们的客服团队将在24小时内回复您的请求。
解决问题
根据提供的信息，我们将采取相应措施解决您的问题。
