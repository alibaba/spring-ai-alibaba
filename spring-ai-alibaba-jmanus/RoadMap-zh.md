# 我想贡献一点力量，怎么做？
欢迎
1. 请直接fork一个自己的分支
2. 改动好功能，然后提交到自己的分支，然后再pull request
3. github 合并前会有一些检查，主要是check style和标题的。 也需要你签署cla协议。
4. 只要提交，我们就一定会回应的，大胆提交吧！


但我们不做认领机制，做完了我们接受的话，你的名字会被铭记在功能列表里。：）
